# BrandX | Cinematic Dark Mode Landing Page

A high-end, cinematic dark mode landing page designed for premium service brands. Featuring deep navy aesthetics, vibrant cyan and magenta glows, and immersive interactive elements.

![BrandX Preview](https://via.placeholder.com/1200x600/0a0e27/00d9ff?text=BrandX+Premium+Landing+Page)

## ✨ Features

- **Cinematic Dark Luxury Aesthetic**: Deep space theme with glassmorphism and subtle glows.
- **Interactive Grid System**: Asymmetric service tiles that expand on click to reveal detailed content.
- **Custom Dual-Element Cursor**: Smooth lagging ring and center dot that reacts to interactive elements.
- **Staggered Reveal Animations**: Hero headlines with character-by-character flip-in animations.
- **Parallax Background**: Floating particle system that reacts to scroll depth.
- **Process Visualization**: Vertical timeline for desktop that adapts for mobile.
- **Fully Responsive**: Optimized for all screen sizes from mobile to wide desktops.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox/Grid, backdrop-filters, and keyframe animations.
- **JavaScript (Vanilla)**: Interaction logic, custom cursor, and scroll observers.
- **Phosphor Icons**: Premium, clean iconography.

## 🚀 Getting Started

### Prerequisites

No build tools are required. You just need a modern web browser.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/balaji3245/brandx-info.git
   ```
2. Open `index.html` in your browser or use a live server extension.

## 🎨 Design Specifications

- **Primary Background**: `#0a0e27`
- **Accent Colors**: `#00d9ff` (Cyan), `#ff00ff` (Magenta), `#0066ff` (Blue)
- **Typography**: 
  - Display: *Space Grotesk* (Bold, 700+)
  - Body: *Outfit* (Light/Regular, 1.6 line-height)
